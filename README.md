# WeatherForecaster Model Z🌦️
I created this open-source project using the newest Python model. I donwloaded it's pacakge in the terminal and got to work by creating an amazing weather forecasting project! And the crazy thing is, I didn't even need an API key!

# What sources did you use?⁉️
As for sources, I used OpenMeteo, a free weather recording and forcasting service running for almost a century! I created my project using PyCharm and deployed it using Streamlit.

# How do I use it?🧑‍💻
Using WeatherForecaster is quite easy. All you have to do is enter the name of a city (ex. Chicago or Los Angeles) and press enter. The website brings info using JSON from the data source and comes back as a weather module. The frontend features a 7 day forecast for the city in °C. It also has an interactive temperature trend graph and extra features such as sharing in the top-right corner.

# Is there anything else I should know?🚀
As of now, WeatherForecaster is available to record the data of many cities all around the world. However, some remote places might not have the proper weather detection services. So if you can't find a weather forecast for a certain city, please use an online resource instead.
